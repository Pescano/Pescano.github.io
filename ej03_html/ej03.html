Definiciones

Hardware

Es la parte tangible de cualquier sistema informatico. Esta puede ser desde la caja hasta la placa base.

Software

Es la parte lógica de un sistema informático. Esta puede ser desde los programas hasta el sistema operativo.

Sistema operativo

Es el conjunto de programas de un sistema informático que gestiona los recursos de hardware y provee servicios
los programas de aplicación de software.

Lenguaje de programación

Un lenguaje de programación es un lenguaje formal que le proporciona a una persona, en este caso el programador,
la capacidad de escribir (o programar) una serie de instrucciones o secuencias de órdenes en forma de algoritmos
con el fin de controlar el comportamiento físico o lógico de un sistema informático, de manera que se puedan
obtener diversas clases de datos o ejecutar determinadas tareas.
